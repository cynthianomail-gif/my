Slot Overview（來源：BWB + SlotCatalog + iGamingToday）

Truth 是一款陰謀論諷刺主題老虎機，玩家置身於一個塞滿機密文件的昏暗辦公室。盤面為 5×5，15 條固定 Paylines，下注範圍 $/€0.10–€50，RTP 96.1%，高波動。Max Win 20,000×。

圖示說明：低值圖示為金字塔、腳掌、噴射燃料筒、文件夾（5 OAK 賠 1×–3×）；高值 5 個角色圖示（5 OAK 賠 7.5×–50×）；Wild 可在所有列出現，替換所有一般圖示，5 OAK 賠 50×。

4 種隨機 Conspiracy Features

每局隨機觸發以下其中一種：

Birds Aren't Real（鳥類不是真的） — 無人機鳥群飛過盤面，隨機位置落下並「故障墜毀」，留下 Wild 圖示。

5G Transmission（5G 傳播） — 盤面出現攜帶 ×2–×5 乘數的 5G Wild，每轉可隨機複製自身到空格或將乘數傳送給鄰近 Wild，直到無法啟動為止。

The Reptilians Are Everywhere（爬蟲人無處不在） — 隨機選擇一種角色圖示（M1–M4），嘗試將所有相同圖示全部轉換為最高值角色（Reptilian H1）。

It's All a Simulation（這一切都是模擬） — 重新模擬當前盤面，套用更有利的圖示分布，可能直接升級為更大贏線。

Bonus Game（Truth Realm）

落地 3 個 Scatter 觸發 Bonus。使用 Truth Realm 機制——盤面特定區域轉換，Conspiracy Features 觸發頻率大幅提升，Wild 覆蓋率增加，形成多輪連鎖爆發。

Super Bonus Game

落地 4+ 個 Scatter 進入 Super Bonus，所有 Conspiracy Features 同時啟動，Wild 和乘數覆蓋整個盤面，是最高潛力模式。

The Store（購買系統）

內含多種 Booster、Bonus Buy、Highlight Reels（播放若干轉取最高的3局）、Loot Box（Bronze/Silver/Gold 等級的 Enhanced Bonus）。Bronze Bonus (128×) RTP 96.6%；各級 Enhanced Bonus 有不同的 Star Level 決定獎勵規模。
